#### Technologies Used

```
ReactJS on client side
NodeJS for Rest API
Sequelize ORM for MySQL database
Stripe for Payment
```

#### How to run locally

```
Open 3 terminals

First terminal:
    cd backend
    npm i
    npm start

Second terminal:
    cd client
    npm i
    npm start

Third terminal:
    cd populate_database
    npm i
    npm start
```
